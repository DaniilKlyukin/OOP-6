# Объектно-ориентированное программирование 2 (6 семестр)

Учебный репозиторий для студентов 3 курса (6 семестр) направления **«Прикладная математика и информатика»**. Курс посвящен продвинутому проектированию корпоративного ПО, изучению архитектурных паттернов (GoF), созданию современных Desktop (Avalonia UI) и Web (ASP.NET Core) приложений, а также принципам Чистой архитектуры (Clean Architecture).

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

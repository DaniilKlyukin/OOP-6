# Курс "Объектно-ориентированное программирование 2" (6 семестр)

Данный репозиторий содержит учебные материалы для студентов 6 семестра направления "Прикладная математика и информатика". Курс посвящен профессиональной разработке программного обеспечения: изучению архитектурных паттернов, принципов чистой архитектуры, современных фреймворков для создания Desktop (Avalonia UI, WPF) и Web (ASP.NET Core) приложений.

## 📚 Содержание курса

### Лекционные материалы
Лекции охватывают полный цикл разработки от проектирования до развертывания.

- [Лекция 1](Лекции/ООП%202%20лек%201.pptx) - Классификация ПО, клиент-серверное взаимодействие, фундаментальные принципы проектирования (KISS, DRY).
- [Лекция 2](Лекции/ООП%202%20лек%202.pptx) - Эволюция Desktop-технологий: Windows Forms, WPF, Avalonia. Язык XAML. Архитектурный паттерн MVVM, реализация `INotifyPropertyChanged` и `ICommand`.
- [Лекция 3](Лекции/ООП%202%20лек%203.pptx) - Классификация паттернов GoF. Порождающие (Singleton, Factory Method) и структурные (Decorator, Adapter, Facade) паттерны.
- [Лекция 4](Лекции/ООП%202%20лек%204.pptx) - Поведенческие паттерны GoF: Strategy, Observer, Command, State.
- [Лекция 5](Лекции/ООП%202%20лек%205.pptx) - Введение в ASP.NET Core. Протокол HTTP, паттерн MVC, маршрутизация, контроллеры и представления (Razor).
- [Лекция 6](Лекции/ООП%202%20лек%206.pptx) - ORM и Entity Framework Core. Проектирование моделей, `DbContext`, миграции, выполнение CRUD-операций через LINQ.
- [Лекция 7](Лекции/ООП%202%20лек%207.pptx) - Инверсия управления (IoC) и внедрение зависимостей (DI). DI-контейнер, жизненные циклы сервисов, внедрение в контроллеры ASP.NET Core.
- [Лекция 8](Лекции/ООП%202%20лек%208.pptx) - Чистая архитектура (Clean Architecture). Слои приложения, разделение моделей (Domain, Entity, DTO). REST API, Swagger, контейнеризация с Docker.

### Лабораторные работы
Комплексные задания, направленные на создание профессиональных приложений с использованием изученных технологий и паттернов.

| № | Тема | Задание |
|---|------|---------|
| 1 | [Реализация Desktop-приложения на Avalonia UI / WPF с MVVM](Лабораторные/Лабораторная%20работа%201.docx) | Создание приложения по варианту с разделением на Model, View, ViewModel, использованием команд (`ICommand`) и коллекций (`ObservableCollection`). |
| 2 | [Применение паттернов GoF](Лабораторные/Лабораторная%20работа%202.docx) | Интеграция паттернов Singleton, Strategy, State, Decorator и Factory в Desktop-приложение с визуальной демонстрацией их работы. |
| 3 | [Разработка Web-приложения "Каталог" на ASP.NET Core MVC с EF Core](Лабораторные/Лабораторная%20работа%203.docx) | Реализация полноценного веб-сайта с CRUD-операциями, поиском и стилизацией с использованием Entity Framework Core (Code-First, миграции). |
| 4 | [Реализация веб-сервиса на основе Чистой архитектуры и DI](Лабораторные/Лабораторная%20работа%204.docx) | Создание многослойного решения (Domain, Application, Infrastructure, API) с паттерном Repository, внедрением зависимостей, Swagger и Docker. |

### Практические работы
Проектные работы, направленные на проектирование архитектуры и аудит кода.

| № | Тема | Задание |
|---|------|---------|
| 1 | [Проектирование архитектуры Desktop-приложения на основе MVVM](Практики/Практическая%20работа%201.docx) | Разработка архитектуры (Model, ViewModel, эскиз интерфейса, схема взаимодействия) для заданной предметной области. |
| 2 | [Архитектурное проектирование с использованием паттернов GoF](Практики/Практическая%20работа%202.docx) | Подбор паттернов для трех бизнес-кейсов, построение UML-диаграмм и обоснование архитектурных решений. |
| 3 | [Проектирование REST API и схемы реляционной базы данных](Практики/Практическая%20работа%203.docx) | Создание спецификации API (эндпоинты, DTO, карта ошибок) и схемы БД (минимум 3 связанные таблицы). |
| 4 | [Архитектурный аудит и проектирование рефакторинга в стиле Clean Architecture](Практики/Практическая%20работа%204.docx) | Анализ "плохого" кода, выявление нарушений принципов SOLID и проектирование перехода к многослойной архитектуре с DI. |

### Экзаменационные материалы
- [Билеты к экзамену](Билеты.docx) - 17 экзаменационных билетов, включающих теоретические вопросы (EF Core, паттерны GoF, MVC/MVVM, IoC/DI, ASP.NET Core, REST, Clean Architecture) и практические задания по разработке приложений.

## 🎯 Цели курса

- Освоение профессиональных паттернов проектирования (GoF) и архитектурных стилей (MVVM, MVC, Clean Architecture).
- Изучение современных фреймворков для создания Desktop (Avalonia UI) и Web (ASP.NET Core) приложений.
- Получение навыков работы с базами данных через ORM (Entity Framework Core).
- Понимание и применение принципов инверсии управления и внедрения зависимостей.
- Освоение инструментов профессиональной разработки: Swagger, Postman, Docker.
- Развитие навыков архитектурного мышления и критического анализа кода.

## 📋 Требования к выполнению работ

1.  Работы выполняются в среде Visual Studio 2026 / JetBrains Rider.
2.  Код должен соответствовать принципам чистой архитектуры и содержать XML-комментарии.
3.  Для Desktop-приложений обязательно использование паттерна MVVM и платформы Avalonia UI / WPF.
4.  Для веб-приложений обязательна настройка Swagger и, для высоких оценок, контейнеризация с Docker.
5.  Отчеты по лабораторным работам оформляются в соответствии с методическими указаниями.
6.  Работы сдаются в установленные сроки.
